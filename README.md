# Full Stack Student Management System

## 📌 Project Description

This is a full stack Student Management System developed using:

* Spring Boot (Backend)
* React (Frontend)

It performs CRUD operations like:

* Add Student
* View Students
* Update Student
* Delete Student

---

## ⚙️ Technologies Used

* Java (Spring Boot)
* React JS (Vite)
* MySQL (or H2)
* REST API
* Maven

---

## 📁 Project Structure

* backend → Spring Boot Application
* frontend → React Application

---

## ▶️ How to Run

### Backend

1. Open backend folder in Eclipse / STS
2. Run BackendApplication.java

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🌐 API Endpoints

* GET /students
* POST /students
* PUT /students/{id}
* DELETE /students/{id}
